3.5 DSSD front-back Correlation(II)¶

目的：¶

  • 对x-y的所有条进行能量刻度 (幅度归一化)
  • 学习进行复杂分析的思路和方法

3.4节回顾¶

  • 粒子在入射位置为：xstrip=i条、ystrip=j条，标记为$(i,j)$

  • 对于入射到 $(i,j)$ ，能量沉积为E的粒子: $A_i=k_{ij}A_j+b_{ij} $

幅度归一化的目标是，将所有条相对于某一参考条$i$条进行刻度。¶

方法1¶

  • 固定参考条$i$条不变，按照上述方程依次遍历所有的$j$条, 得到y面的归一化系数：$\lbrace k_{i,j},b_{i,j}\rbrace \quad ,j=0,1,...N_j $
  • 对y面所有条进行刻度：$\lbrace A'_j=k_{i,j}A_j+b_{i,j} \rbrace\quad ,j=0,1,...N_j$。
  • 以刻度后y面某一条$j$条作为参考条，对所有x面的条进行刻度， $ A'_{j}=k_{j,i}A_i+b_{j,i}$

经上述步骤，DSSD的$x$面和$y$面的所有条能量都归一化到第$i$参考条

  • 该方法利用的是每个x条和每个y条重叠区域事件(pixel),前提是入射到每一个$(i,j)$ pixel的粒子有足够的统计。

方法2¶

  • 更充分利用x-y面的关联关系
  1. 在x面和y面分别寻找统计最高的条$(i_m,j_m)$
  1. 以$j_m$为参考条，刻度y面一系列条： $i_{m-k_1},. ,i_{m},.,i_{m+k_2}$
  1. 以刻度后的$i_{m-k_1},. ,i_{m},.,i_{m+k_2}$ 为参考条，刻度 $j_{m-k_3},.,j_{m},.,j_{m+k_4}$
  1. 按照上述步骤向外推进，直到最外边缘的条的正背面关联有足够的统计做拟合。

for(int ix=xrange[0]; ix<=xrange[1]; ix++) {
    grxc[ix]=new TGraph;
    int ng=0;
    //fit with pol1 first, make new TGraph for the next fitting..
    if(grx[ix][ys]->GetN()>20) {//only fit the graph with enough statistics
        grx[ix][ys]->Fit(fpa1,"Q ROB");
        fpa1->GetParameters(&par[0]);
        for(int k=0; k<grx[ix][ys]->GetN(); k++) {
            double x=grx[ix][ys]->GetX()[k];
            double y=grx[ix][ys]->GetY()[k];
            if (abs(y-(par[0]+par[1]*x))<20) {
                grxc[ix]->SetPoint(ng,x,y);
                ng++;
            }            
        }
    }
    //fit with pol2
    if(grxc[ix]->GetN()>20) {
        grxc[ix]->Fit(fpa2,"Q");
        fpa2->GetParameters(&par2[ix][0]);
        chi2ndf[ix]=fpa2->GetChisquare()/fpa2->GetNDF();
        hresname.Form("hres%02d",ix);
        hres[ix] = new TH2F(hresname,hresname,100,-50,50,1000,200,8000);
        xmax[ix]=0;
        for(int j = 0;j < grx[ix][ys]->GetN(); j++){
            double x=grx[ix][ys]->GetX()[j];
            double y=grx[ix][ys]->GetY()[j];
            if(x>xmax[ix]) xmax[ix]=x;
            x = y-(par2[ix][0]+par2[ix][1]*x+par2[ix][2]*x*x);
            hres[ix]->Fill(x,y);
            
        }
        Ncx[ix]=grxc[ix]->GetN();
        sout.Form("%4d%9.2f%11.6f%20e%16.2f%8d%8d"  ,ix  ,par2[ix][0]  ,par2[ix][1]  ,par2[ix][2]  ,chi2ndf[ix], xmax[ix],Ncx[ix]);
        cout<<sout<<endl;
    }
}
